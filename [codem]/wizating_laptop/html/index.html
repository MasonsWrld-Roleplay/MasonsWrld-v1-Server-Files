fTractionCurveMax ( 0 - 3.7 )

Cornering grip of the vehicle as a multiplier of the tire surface friction.

fTractionCurveMin ( 0 - 3.5 )

Accelerating/braking grip of the vehicle as a multiplier of the tire surface friction.

fTractionLossMult ( 0 - 1.4 )

Affects how much grip is changed when driving on asphalt and mud
-Higher values make the car less responsive and prone to sliding.

fTractionCurveLateral ( 1 - 120 )

Shape of lateral traction curve.
- The degress of angle before the car loses grip.

fTractionSpringDeltaMax ( 0.02 - 0.5 )

This value determines at what distance above the ground the car will lose traction.

fTractionBiasFront ( 0.01 - 0.99 )

Determines the distribution of traction from front to rear.
0.01 = only rear axle has traction.
0.99 = only front axle has traction.
0.5 = both axles have equal traction.
Entering a value of 0.0 or 1.0 causes the vehicle to have no traction.

fLowSpeedTractionLossMult ( 0 - 2.2 )

How much traction is reduced at low speed.
0.0 means normal traction. It affects mainly car burnout when accelerating.
Decreasing value will cause less burnout, less sliding at start.
Higher value will cause more burnout.

fInitialDriveForce ( 0 - 3 )

This modifies the game's calculation of drive force (from the output of the transmission).
1.0 uses drive force calculation unmodified.
Values less than 1.0 will in effect give the vehicle less drive force.
Values greater than 1.0 will produce more drive force.

fDriveInertia ( 0.1 - 2.5 )

Describes how fast an engine will rev.
Default value is 1.0, (or no modification of drive intertia).
Bigger values = quicker Redline (maximum engine speed)

fClutchChangeRateScaleUpShift ( 0.3 - 9 )

Clutch speed multiplier on up shifts.
Bigger numbers = faster shifts.

fClutchChangeRateScaleDownShift ( 0.3 - 9 )

Clutch speed multiplier on down shifts.
Bigger numbers = faster shifts

fInitialDragCoeff ( 0.9 - 300)

Sets the drag coefficient of the vehicle.
Increase to simulate aerodynamic drag.

fInitialDriveMaxFlatVel ( 100 - 330 )

Determines the speed at redline in top gear
Controls the final drive of the vehicle's gearbox.
Multiply the number by 0.82 to get the speed in mph
Multiply by 1.32 to get kph.
Setting this value does not guarantee the vehicle will reach this speed.

fDriveBiasFront ( 0 - 1 )

This value is used to determine whether a vehicles drive.
Value: 0.0 is rear wheel drive
1.0 is front wheel drive
Any value between 0.01 and 0.99 is four wheel drive
(0.5 give both front and rear axles equal force, being perfect 4WD.)

fBrakeForce ( 0.001 - 5 )

Multiplies the game's calculation of deceleration.
Bigger number = harder braking.

fHandBrakeForce ( 0.001 - 5 )

Braking power for handbrake.
Bigger number = harder braking.

fBrakeBiasFront ( 0 - 0.8)

This controls the distribution of braking force between the front and rear axle.
0.0 means the rear axle only receives brake force
1.0 means the front axle only receives brake force
0.5 gives both axles equal brake force.

fSteeringLock ( 0 - 75)

Multiplier of the game's calculation of the angle a steer wheel will turn while at full turn.
Steering lock is directly related to turning radius.

fSuspensionForce ( 0 - 9 )

Affects how strong suspension is.
Can help if car is easily flipped over when turning.
Lower limit for zero force at full extension.

fSuspensionCompDamp ( 0 - 8)

Damping during strut compression.
Bigger = stiffer

fAntiRollBarForce ( 0 - 5)

The spring constant that is transmitted to the opposite wheel
under compression larger numbers are a larger force.
Larger Numbers = less body roll

fAntiRollBarBiasFront ( 0 - 1)

The bias between front and rear for the antiroll bar.
0 front, 1 rear

fSuspensionUpperLimit ( 1 - 0)

How far can wheels move up from original position.

fSuspensionLowerLimit ( -1 - 0)

How far can wheels move down from original position.

fRollCenterHeightFront ( -1 - 1.2)

The roll center height for the front axle.
High values place the roll center closer to the center of mass, reducing body roll.
Values placing the roll center higher than the center of mass will induce negative body roll.
Low values increase the roll center distance from the center of mass, increasing body roll.
Values placing the roll center too far from the center of mass may increase rollover chances.

fRollCenterHeightRear ( -1 - 1.2)

The roll center height for the rear axle.
High values place the roll center closer to the center of mass, reducing body roll.
Values placing the roll center higher than the center of mass will induce negative body roll.
Low values increase the roll center distance from the center of mass, increasing body roll.
Values placing the roll center too far from the center of mass may increase rollover chances.

fSuspensionReboundDamp ( 0 - 10.8)

Damping during strut rebound.
Bigger = stiffer.

fSuspensionRaise ( 0.085 - 0.35)

The amount that the suspension raises the body off the wheels.
Recommend adjusting at second decimal unless vehicle has room to move.
ie -0.02 is plenty of drop on an already low car.

fSuspensionBiasFront ( 0 - 0.8)

Force damping scale front/back.
If more wheels at back (e.g. trucks) need front suspension to be stronger.
This value determines which suspension is stronger, front or rear.
If value is above 0.50 then front is stiffer, when below, rear.
